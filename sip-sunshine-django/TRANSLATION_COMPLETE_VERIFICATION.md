## Translation System Features

### 1. **Template Tags** ({% trans %} tags)
All strings wrapped with `{% trans %}` for automatic translation

### 2. **Django i18n**
- Enabled in settings
- Middleware configured
- Locale paths set

### 3. **django-parler Integration**
- Database models support multiple languages
- Admin interface shows language tabs
- TranslatedFields for dynamic content

### 4. **Language Fallback**
- Default fallback: English
- If translation missing, falls back to English
- No broken strings or empty placeholders

### 5. **Language Switcher**
- Available in website header
- Shows all 3 languages
- Reloads page in selected language

## Admin Translation Interface

### How to Add Translations via Admin

1. **Login**: `admin / admin123456`
2. **Go to**: Restaurant > Menu Items (or Pages, Blog Posts)
3. **Select item** to edit
4. **Use language tabs**: English | Dutch | Français
5. **Edit translations** for each language
6. **Save**
